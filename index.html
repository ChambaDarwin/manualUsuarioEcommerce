Mensajes y Comentarios

En esta sección puedes compartir tus comentarios y sugerencias. Ten en cuenta que tu nombre de usuario será visible para el administrador cuando revise tus mensajes. Valoramos tu opinión y nos esforzamos por mejorar continuamente basados en tus comentarios.

Historial de Transacciones

Consulta tu historial completo de tickets de venta. Puedes buscar transacciones por fecha específica, imprimir tickets anteriores en caso de pérdida y revisar todas tus operaciones realizadas en el sistema. Toda tu actividad comercial queda registrada para futuras consultas.

Cerrar Sesión

Al seleccionar esta opción, serás redirigido fuera de la aplicación. No te preocupes, toda la información de tus transacciones permanecerá segura y accesible en la sección de Historial para cuando necesites consultarla nuevamente.

Tengo otro código

Si dispones de un código diferente o deseas explorar productos de otros proveedores, ingresa el nuevo código en este apartado. El sistema te mostrará automáticamente los productos y servicios disponibles del administrador asociado a ese código.

Solicitar Permisos de Administrador

Para acceder a todas las funcionalidades avanzadas de la aplicación, primero debes solicitar permisos de administrador. Este proceso puede requerir verificación y aprobación, por lo que el acceso no es inmediato. Una vez concedidos los permisos, la opción cambiará a "Panel de Administrador", dándote acceso a:

  • Configuración de impresoras térmicas (55mm y 80mm)
  • Gestión de productos (agregar, editar, eliminar)
  • Visualización de comentarios de usuarios
  • Personalización de tickets de venta
  • Restablecimiento de contadores
  • Reportes de ventas por fecha y rangos específicos

Eliminar Cuenta

En esta sección puedes eliminar permanentemente tu cuenta de administrador. Ten en cuenta que esta acción eliminará toda la información asociada a tu cuenta de forma irreversible. Te recomendamos asegurarte de tener respaldos de tu información antes de proceder con la eliminación.

Funcionalidades de Administrador

Configuración de Impresora

Para utilizar la funcionalidad de impresión, primero debes aceptar los permisos correspondientes. Si los deniegas, no podrás utilizar la impresora. Sigue estos pasos:

  1. Empareja tu impresora Bluetooth con el dispositivo móvil
  2. Selecciona la impresora desde la lista de dispositivos disponibles
  3. Espera a que se establezca la conexión (un indicador de color te confirmará el estado)
  4. Configura el tipo de impresora (55mm u 80mm) según tu hardware
  5. Realiza una prueba de impresión para verificar la configuración

Asegúrate de tener papel en la impresora antes de realizar cualquier prueba o impresión.

Agregar Productos

Sigue este proceso para añadir nuevos productos al catálogo:

  1. Imagen (opcional): Añade una imagen para hacer tus productos más atractivos
  2. Nombre para ticket: Define el nombre que aparecerá en los tickets, considerando el límite de caracteres según tu tipo de impresora (55mm u 80mm)
  3. Detalles adicionales (opcional): Proporciona información extendida que se mostrará en la ficha del producto
  4. Texto para búsqueda por voz: Añade términos alternativos para facilitar la búsqueda vocal
  5. Precio: Establece el valor del producto
  6. Categoría: Asigna hasta 8 categorías diferentes para organizar tus productos
  7. Subcategoría: Crea hasta 3 subcategorías por categoría para una organización más detallada
  8. Finaliza presionando el botón "Registrar"

Productos Registrados

Consulta y gestiona todos tus productos registrados. Puedes editar información, eliminar productos obsoletos o buscar elementos específicos por nombre. Para inventarios extensos, utiliza las opciones de filtrado y búsqueda avanzada para localizar productos rápidamente.

Comentarios de Usuarios

Revisa los comentarios y opiniones que los usuarios han compartido sobre tus productos y servicios. Puedes gestionar estos comentarios según sea necesario, con la opción de eliminar aquellos que no sean apropiados.

Configuración de Ticket de Venta

Personaliza los tickets de venta según tus necesidades:

  1. Establece un código único para que tus clientes accedan a tus productos
  2. Configura la información empresarial que aparecerá en el ticket: nombre del negocio, dirección, teléfono y horario de atención
  3. Selecciona los elementos específicos que deseas incluir en el ticket
  4. Define un número inicial para el contador de tickets
  5. Restablece el contador cuando sea necesario

Resumen de Caja

Consulta reportes detallados de ventas por fecha o rangos personalizados. Visualiza los productos más vendidos, el total de ingresos y obtén perspectivas valiosas sobre el desempeño comercial de tu negocio.

Carrito de Compra

Al presionar este botón, activarás la función de búsqueda por voz. Primero indica la cantidad deseada y luego el nombre del producto. Por ejemplo: "una computadora, dos teclados y un mouse". Finaliza la grabación presionando nuevamente el botón. El sistema procesará tu solicitud y agregará los productos al carrito.

Presiona este botón para confirmar y agregar los productos seleccionados al carrito. Desde aquí puedes modificar cantidades, eliminar productos o continuar shopping. Una vez impreso el ticket, el carrito se vaciará automáticamente.

Añade descripciones personalizadas al ticket de venta. Presiona el botón, graba tu mensaje y vuelve a presionar para finalizar. Puedes editar o eliminar la descripción antes de confirmar con . La descripción aparecerá al final del ticket.

Alternativamente, puedes buscar productos manualmente en la pantalla principal o utilizando la lupa de búsqueda para localizar productos por nombre. Finaliza el proceso presionando el botón de impresión para generar el ticket y completar la transacción.
